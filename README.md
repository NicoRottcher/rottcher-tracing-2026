# Code repository for: Tracing Dissolved Iridium in Proton Exchange Membrane Water Electrolysis Using Operando Hard-X-Ray Fluorescence 

This is the code repository for the accompanying research article: Nico C. Röttcher, Lukas Löttert, Michael Buchhorn, Martha Y. Stando, Yi-Hsuan Wu, Andreas Körner, Simon Thiele, Karl J.J. Mayrhofer, Andreas Hutzler, Serhiy Cherevko, Dominik Dworschak, Jakub Drnec, Anna T.S. Freiberg, **2026**, _submitted_.

After acceptance, the repo will include:
- Instruction for locally setting up the repo
- Script to fetch raw data from corresponding Zenodo data repository
- Data analysis routines
- Visualization routines for the figures of the manuscript and supplemental information

Raw data from WAXS 2D-detector are accessible in the ESRF data repository: [10.15151/ESRF-DC-2434918683](https://doi.esrf.fr/10.15151/ESRF-DC-2011729981)
